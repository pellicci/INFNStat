
# Exercises for the INFN School of Statistics 2026

These exercises use CMS Run‑1 public data (~25 fb⁻¹) to study the Higgs boson in the four‑lepton final state using RooFit and RooStats.

![m4l spectrum](assets/m4l_spectrum.gif)

## Resources

The original course includes RooFit, RooStats, and statistical inference exercises covering:

- Maximum likelihood fits
- p-values and significance
- Confidence intervals
- Bayesian inference
- CLs upper limits
- Toy Monte Carlo studies

Source code is available in the `code/` directory.
